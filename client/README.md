# RM Client
Telegram WebApp UI (Canvas/WebGL, mobile-portrait).  
Пока заготовка. Реализацию добавим отдельным шагом через Cursor.

## Feedback System
- Обработка `hit_feedback` событий от сервера.
- HUD обновляется мгновенно.
- Эффекты: всплывающий текст, вспышка, screen shake, звук, вибрация.
- Вибрация — только у пострадавшего.
- Настройки эффектов управляются пользователем.
