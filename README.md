# Рейтинг микрозаймов на карту

Подборка микрозаймов для тех, кому нужны деньги до зарплаты, первый займ без процентов, новые МФО, круглосуточные займы и варианты без лишних платных подписок.

На этой витрине собраны тематические разделы, чтобы было проще выбрать подходящий вариант под конкретную ситуацию.

## Основные разделы

- [Рейтинг микрозаймов](docs/index.html)
- [Деньги до зарплаты](docs/dengi-do-zarplaty.html)
- [Первый займ без процентов](docs/pervyi-zaim-bez-procentov.html)
- [Новые МФО](docs/novye-mfo.html)
- [Малоизвестные МФО](docs/maloizvestnye-mfo.html)
- [Микрозаймы без подписок](docs/bez-podpisok.html)

## Что есть на сайте

- подборки МФО по разным сценариям;
- карточки с базовыми условиями;
- быстрые переходы на офферы;
- удобная навигация между разделами.

## Для кого эта витрина

Этот сайт подойдёт тем, кто:

- ищет короткий займ на небольшой срок;
- хочет посмотреть предложения для новых клиентов;
- сравнивает разные МФО перед оформлением;
- хочет найти дополнительные варианты среди новых или менее известных сервисов.

## Как пользоваться

Откройте нужный раздел, сравните предложения и перейдите к выбранному офферу. Для удобства используются короткие ссылки вида `212.su/название-мфо`.

## Важно

Перед оформлением займа важно внимательно читать условия, сроки возврата, итоговую стоимость и возможные дополнительные услуги на стороне МФО.
