# Рейтинг микрозаймов — SEO-витрина под GitHub Pages

Этот репозиторий — основа для SEO-витрины микрозаймов с публикацией через **GitHub Pages** и отдельным слоем редиректов на домене **212.su**.

## Что здесь будет

- индексируемая витрина под запросы про микрозаймы;
- страницы-кластеры под long-tail семантику;
- единая структура карточек МФО;
- ссылки на офферы через `212.su/<slug>`;
- возможность быстро менять CPA-сеть без правки SEO-страниц.

## Базовая модель

- **GitHub Pages** = витрина и контент
- **212.su/slug** = редиректы на офферы
- **Фокус** = только микрозаймы
- **Подход** = не бить в слишком широкие запросы, а собирать long-tail и тематические кластеры

## Стартовые разделы

- Рейтинг микрозаймов
- Деньги до зарплаты
- Первый займ без процентов
- Новые МФО
- Малоизвестные МФО
- Круглосуточные микрозаймы
- Микрозаймы без звонков
- Микрозаймы с 18 лет
- Микрозаймы без подписок

## Правила проекта

1. Не смешивать SEO-страницы и редиректы.
2. Все кнопки офферов вести через `212.su/<slug>`.
3. Одна страница = один основной интент и близкий кластер.
4. Не распыляться на кредитки, банки и долгосрочные займы.
5. На каждой странице должны быть:
   - сильный `H1`
   - краткое интро
   - блок сравнения / карточки МФО
   - кому подойдет
   - как отбирали
   - FAQ
   - дата обновления
   - перелинковка на соседние разделы

## Следующие шаги

1. Включить GitHub Pages из папки `docs/`.
2. Привязать поддомен витрины.
3. Добавить реальные карточки МФО и slug-редиректы на `212.su`.
4. Обновлять страницы каждые 7–14 дней.

## Рабочий принцип по офферам

SEO-страницы не должны ссылаться напрямую на CPA-сеть. Для каждого оффера используется короткий адрес:

- `212.su/zaimer`
- `212.su/webbankir`
- `212.su/ezaem`

Это позволяет безболезненно менять партнёрскую ссылку на стороне хостинга.
